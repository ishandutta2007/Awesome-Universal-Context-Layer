# Awesome-Universal-Context-Layer
## Top Universal Context Layers Ecosystem

**Curated List of SaaS Products & Open-Source GitHub Projects**  
*Focused on Semantic Layers, Unified Data Context & AI Knowledge Access*  
**Last updated: March 2026**

This repository tracks notable **SaaS platforms** and **open-source projects** building **Universal Context Layers**. These tools provide a unified semantic layer over disparate data sources, enabling consistent metrics, business context, knowledge graphs, and AI-ready access for analytics, agents, and applications.

**Examples** include Unabyss, Atlan, Snowflake Horizon Context, dbt Semantic Layer, and Attio (the category leaders). Tools listed here emphasize **semantic consistency**, governance, AI agent integration, and real-time context delivery.

**Open-source emphasis**: This section is heavily expanded with every major active project for self-hosting, local deployment, full customization, and complete data control — ideal for data teams and AI engineers who want sovereign, transparent context layers without vendor lock-in.

Contributions welcome! Open a PR to add/update entries. Keep descriptions factual and link to official sites.

## Open-Source GitHub Projects

### Dedicated Universal Context & Semantic Layer Projects

- **[dbt Core](https://github.com/dbt-labs/dbt-core)**  
  The leading open-source semantic modeling tool. Enables consistent, reusable business logic and metrics across warehouses and AI applications.

- **[Cube](https://github.com/cube-js/cube)**  
  Open-source semantic layer and API platform for building consistent data access layers with caching and multi-backend support.

- **[DataHub](https://github.com/datahub-project/datahub)**  
  Popular open-source metadata platform with strong semantic context, lineage, and AI-ready knowledge graph features.

- **[Amundsen](https://github.com/amundsen-io/amundsen)**  
  Open-source data discovery and metadata engine with semantic search and context capabilities.

- **[Trino](https://github.com/trinodb/trino)** (formerly PrestoSQL)  
  Distributed SQL query engine with excellent support for building universal semantic layers across multiple data sources.

- **[Apache Iceberg](https://github.com/apache/iceberg)**  
  Open table format with powerful semantic capabilities for data lakes and consistent querying.

- **[LlamaIndex](https://github.com/run-llama/llama_index)**  
  Leading open-source framework for building context and knowledge layers for LLM agents with advanced indexing and retrieval.

- **[Haystack](https://github.com/deepset-ai/haystack)**  
  Production-ready open-source framework for building semantic search and context layers for AI applications.

- **[Mem0](https://github.com/mem0ai/mem0)**  
  Open-source memory and context management layer for AI agents with long-term semantic recall.

- **[Zep](https://github.com/getzep/zep)**  
  Open-source long-term memory store optimized for building persistent context layers for conversational and agentic systems.

### Additional Strong Open-Source Options

- **[MetricFlow](https://github.com/transform/metricflow)** — Semantic metrics layer from Transform (dbt Labs).
- **[Great Expectations](https://github.com/great-expectations/great_expectations)** — Data quality and semantic validation framework.
- **[OpenMetadata](https://github.com/open-metadata/OpenMetadata)** — End-to-end metadata management with semantic features.
- **[Superset](https://github.com/apache/superset)** — Data exploration with semantic layer integration.
- **[LangGraph Context Agents](https://github.com/langchain-ai/langgraph)** — For building stateful, semantic context-aware multi-agent systems.
- Many community **RAG + Knowledge Graph** stacks for universal AI context layers.

**Frameworks for building custom layers**: Combine **dbt Core**, **Cube**, **DataHub**, and **LlamaIndex** with **LangGraph** and vector databases to create fully open, sovereign universal context layers.
